Search volunteering opportunity


    The search implementation is not so user-friendly
    Enter the field/s by which you want to search for opportunities separated by a
    semi-colon (and optional space/s) in the 1st box.

    Following are the valid search fields : title, location, availability

    Enter the terms for those fields in the respective order of the fields, separated by a
    semi-colon (and optional space/s) in the 2nd box.

    Consider the following example.
      enter ( title; location ) in the first search box without the brackets
      enter ( intern; mumbai ) in the second search box without the brackets

    The above example will show all opportunities with title = intern and location = mumbai
    Note that the search is not case sensitive.
    The search uses regular expressions, so partial matches are also shown.

    Example 1 : For title 'Manager', result will also contain 'Communications Manager' if such title
              exists in the db.
    Example 2 : Also helps to find location better, for location in (city, state) format,
              location can be found by state also)

    Also, please donot type ';' before the first or after the last field/search_terms
    as it causes some things to break due to the way text is processed before searching
